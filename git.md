1. 레포지토리 접속
   - GitHub 접속
   - 본인 레포지토리 들어가기
2. Settings 이동
   - 상단 메뉴에서 Settings 클릭
3. Collaborators (또는 Access) 메뉴
   - 좌측 메뉴에서 Collaborators 또는 Manage access / Access 클릭
4. 사람 초대하기
   - "Add people" 또는 "Invite a collaborator" 클릭
   - 초대할 사람의: GitHub 아이디 또는 이메일 입력
5. 권한 설정 (중요🔥)
   - 초대할 때 또는 초대한 후 권한 설정 가능

- 권한 종류
  - Read 👉 코드 보기만 가능
  - Triage 👉 이슈/PR 관리 가능
  - Write 👉 코드 push 가능 (일반 협업용 👍) <-- 이걸로
  - Maintain 👉 레포 관리 (설정 일부 가능)
  - Admin 👉 모든 권한 (위험, 신중)
